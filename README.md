# robbo
Zabawa programowaniem.

## Kompilacja

Uruchom polecenie:

    $ gprbuild

## Uruchomienie

W pliku `world_1` znajduje się przykładowy świat (`#` oznacza ścianę, ` ` puste miejsce a `^` robota skierowanego na północ). 

Uruchom polecenie:

    $ ./pledge

## Usunięcie zbędnych plików

Uruchom polecenie:

    $ gprclean
